# spectrocloud_cluster_config_template (Resource)

  A resource for creating and managing cluster config templates.

## Example Usage

Basic cluster config template with profiles and policies.

```terraform
resource "spectrocloud_cluster_config_template" "aws_template" {
  name       = "aws-prod-template"
  cloud_type = "aws"

  profiles {
    uid = spectrocloud_cluster_profile.infra.id
  }

  profiles {
    uid = spectrocloud_cluster_profile.addon.id
  }

  policies {
    uid  = spectrocloud_cluster_config_policy.maintenance.id
    kind = "maintenance"
  }
}
```

Minimal cluster config template with just the required fields.

```terraform
resource "spectrocloud_cluster_config_template" "minimal" {
  name       = "minimal-template"
  cloud_type = "azure"
}
```
